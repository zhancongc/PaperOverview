# 高级功能文档 (v4.0)

## 概述

v4.0 版本引入了一系列高级功能，旨在提升生成综述的学术质量和可读性，消除 AI 痕迹。

---

## 1. 自然统计数据嵌入

### 功能描述

将统计数据自然地融入叙述，避免机械的 `(OR=0.65, p<0.001)` 风格引用。

### 问题背景

AI 生成的综述常使用如下引用格式：
```
Zhang等[1](OR=0.65, p<0.001)发现显著效应；Li等[2](OR=1.2, p<0.01)也支持这一结论。
```

这种风格 AI 痕迹明显，每个引用都带统计数据显得机械冗余。

### 解决方案

只对重要发现嵌入数据，数据自然融入结论：

```
关于QFD的实际效果，学界尚未形成完全一致的结论。多数制造业实证研究支持其积极价值，如Zhang等[1]发现实施QFD后产品缺陷率下降了35%。然而，Wang等[2]基于服务企业的研究指出，在组织流程标准化程度较低的情境下，QFD的效果存在明显衰减。
```

### 使用判断规则

| 条件 | 说明 | 示例 |
|------|------|------|
| 突破性发现 | OR<0.5或>2.0, r≥0.7, Cohen's d≥0.8 | 强保护/风险效应 |
| 大样本研究 | n≥1000 | 更有说服力 |
| 高度显著 | p<0.001 | 效应高度显著 |
| 边界条件/负面发现 | 包含"限制"、"衰减"等关键词 | 增加可信度 |
| 常规发现 | 其他情况 | 不使用数据，避免冗余 |

### 代码位置

- 服务类：`services/natural_statistics.py::NaturalStatisticsIntegrator`
- 测试文件：`backend/test_natural_statistics.py`

### 使用示例

```python
from services.natural_statistics import format_natural_citation

paper = {
    "authors": ["Zhang", "Wang"],
    "year": 2021,
    "statistics": {"percentage": 35, "n": 2000}
}

# 自然引用格式
citation = format_natural_citation(
    paper,
    citation_number=1,
    finding_description="研究发现QFD能有效降低产品缺陷率"
)
# 输出: "研究发现QFD能有效降低产品缺陷率，下降幅度达35.0%[1]"
```

## 2. 深度对比分析

### 功能描述

在文献对比中不仅列出对立观点，还追问"这种分歧可能源于..."，展示批判性思维。

### 问题背景

普通综述的对比分析缺乏深度：
```
关于媒体关注的效应，现有研究存在分歧。Zhang等[1]发现显著负相关；
Smith等[2]则发现压力效应；Chen等[3]指出治理水平的调节作用。
```

### 解决方案

添加分歧原因分析，提升学术价值：

```
关于媒体关注的效应，现有研究存在分歧。Zhang等[1]发现显著负相关；
Smith等[2]则发现压力效应；Chen等[3]指出治理水平的调节作用。

> **这种分歧可能源于：**
> - **研究对象差异**：Zhang等[1]和李明等[4]基于中国市场，而Smith等[2]研究美国企业
> - **样本情境差异**：Smith等[2]聚焦于业绩压力情境，而其他研究涵盖一般情境
> - **理论视角差异**：Zhang等[1]强调监督理论，而Smith等[2]关注压力应对理论
```

### 分歧原因类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 样本差异 | 样本量、来源、地域、时间 | n=200 vs n=2000 |
| 方法差异 | 测量方式、分析方法、模型设定 | 实验法 vs 调查法 |
| 情境差异 | 制度环境、市场环境、行业特征 | 发达国家 vs 发展中国家 |
| 理论差异 | 理论视角、假设前提 | 代理理论 vs 权变理论 |
| 数据质量差异 | 数据来源、时效性、准确性 | 自报数据 vs 审计数据 |

### 代码位置

- 服务类：`services/deep_comparison.py::DeepComparisonAnalyzer`
- 格式化类：`services/deep_comparison.py::DeepComparisonFormatter`
- 测试文件：`backend/test_deep_comparison.py`

### 使用示例

```python
from services.deep_comparison import infer_divergence_reasons

paper_a = {"year": 2021, "is_english": False}
paper_b = {"year": 2015, "is_english": True}

reasons = infer_divergence_reasons(paper_a, paper_b, "媒体关注与盈余管理")
# 输出: ['研究对象来自不同市场环境（如中国市场 vs 美国市场）', '研究时间跨度较大（2021 vs 2015）']
```

## 3. 综述润色（消除AI腔）

### 功能描述

自动检测并删除 AI 生成文本中的常见"AI腔"词汇，让语言更干练自然。

### 问题背景

AI 生成的文本包含大量冗余词汇：
```
近年来，随着资本市场的快速发展，媒体治理受到广泛关注。值得注意的是，多项研究发现媒体监督能抑制盈余管理。换言之，媒体具有重要的治理功能。显而易见，这一发现具有重要意义。
```

### 解决方案

删除 AI 腔，精简表达：

```
近五年，媒体治理受到广泛关注。多项研究发现媒体监督能抑制盈余管理，具有重要的治理功能。
```

### AI 腔模式词典

| 类型 | 模式 | 处理方式 |
|------|------|----------|
| 过渡词 | 值得注意的是、显而易见、众所周知 | 删除 |
| 背景铺垫 | 近年来、随着...的发展、在...背景下 | 删除或具体化 |
| 连接词 | 一方面...另一方面、此外、另外 | 删除或用分号代替 |
| 程式化 | 本文旨在、研究结果表明 | 简化 |
| 总结词 | 综上所述、总而言之 | 简化为"综上" |

### 润色方法

| 方法 | 说明 | 压缩率 |
|------|------|--------|
| 规则润色 | 基于25+条规则快速处理 | 30%-40% |
| LLM润色 | 使用大模型智能优化 | 20%-30% |
| 混合模式 | 规则预处理 + LLM精细优化 | 30%-40% |

### 代码位置

- 服务类：`services/review_polisher.py::AIToneEliminator`
- 集成类：`services/review_polisher.py::ReviewPolisher`
- 测试文件：`backend/test_review_polishing.py`

### 使用示例

```python
from services.review_polisher import polish_review_text

ai_text = """
近年来，随着公司治理研究的深入，媒体关注引起了广泛关注。
值得注意的是，多项研究支持了媒体的监督作用。换言之，媒体能有效抑制盈余管理。
"""

polished_text, report = polish_review_text(ai_text, method="rule")
# 输出: "公司治理研究中，媒体关注引起了广泛关注。多项研究支持了媒体的监督作用，媒体能有效抑制盈余管理。"
# 压缩率: 39.6%
```

## 4. 连续引用拆分

### 功能描述

检测连续引用如 `[1-5]`，拆分为结构化陈述，提升可读性。

### 问题背景

连续引用难以识别具体文献：
```
多项研究[1-5]支持QFD的积极作用，指出其能显著提升产品质量。
```

### 解决方案

拆分为具体引用：
```
Zhang等[1]发现QFD能显著降低产品缺陷率；Li等[2]指出QFD能提升客户满意度；
Wang等[3]证实了QFD在制造业的应用价值；Liu等[4]强调组织支持的重要性；
Chen等[5]则发现了QFD的边界条件。
```

### 拆分策略

| 策略 | 说明 |
|------|------|
| 按年份分组 | 早期研究 vs 近期研究 |
| 按主题分组 | 正面发现 vs 负面发现 |
| 按方法分组 | 实证研究 vs 案例研究 |

### 代码位置

- 服务类：`services/citation_splitter.py::CitationSplitter`
- 测试文件：`backend/test_citation_splitter.py`

## 5. 观点碰撞分析

### 功能描述

结构化分析学术争议，提取对立观点并分析分歧原因。

### 输出格式

```markdown
## 争议与对话

### 对立观点A：支持派
- 核心论点：媒体关注能有效抑制盈余管理
- 支持文献：[1-3]

### 对立观点B：反对派
- 核心论点：媒体压力可能诱发盈余管理
- 支持文献：[4-5]

### 可能的原因
1. **样本差异**：支持派研究基于大样本企业，反对派聚焦中小企业
2. **情境差异**：支持派研究涵盖一般情境，反对派聚焦高压情境
3. **理论差异**：支持派强调监督理论，反对派关注压力应对理论
```

### 代码位置

- 分析器：`services/controversy_analyzer.py::ControversyAnalyzer`
- 生成器：`services/controversy_analyzer.py::ControversySectionGenerator`

## 使用建议

### 组合使用

建议按以下顺序组合使用高级功能：

1. **综述生成** → 生成基础综述
2. **自然数据嵌入** → 优化数据引用格式
3. **深度对比分析** → 添加分歧原因分析
4. **观点碰撞分析** → 生成争议章节
5. **连续引用拆分** → 拆分长引用
6. **综述润色** → 消除AI腔

### 参数配置

| 功能 | 关键参数 | 建议值 |
|------|----------|--------|
| 自然数据嵌入 | 数据使用阈值 | n≥1000, p<0.001 |
| 深度对比分析 | 最多分析论文数 | 6篇 |
| 综述润色 | 润色方法 | hybrid |
| 连续引用拆分 | 最大连续长度 | 3 |
