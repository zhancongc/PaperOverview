## 🚀 系统概览

PaperOverview 是一个基于 AI 的智能文献综述生成系统，用户只需输入论文题目，系统即可自动生成包含 50+ 篇参考文献的高质量文献综述。

### 核心功能

| 功能 | 描述 |
|------|------|
| 🔍 智能文献搜索 | 支持中英文文献，语言区分搜索（中文用 AMiner，英文用 OpenAlex） |
| 🔎 查找文献 | **新功能**：独立文献搜索，可预览搜索结果 |
| 💾 论文数据库 | 自动保存所有搜索到的论文，避免重复搜索，提高效率 |
| 📝 生成综述 | 基于 DeepSeek 大模型，自动生成结构化综述 |
| 🎯 智能分析 | **新功能**：题目分析 + 大纲生成 + 关键词提取 |
| ✅ 质量保证 | 自动过滤低质量文献，验证引用数量和年份分布 |
| 📊 参数可配 | 支持自定义文献数量、年份占比、英文占比等 |
| 📈 阶段记录 | **新功能**：追踪每个阶段的执行结果，支持版本对比 |
| 🔄 代码复用 | **新功能**：查找文献和智能分析复用综述生成的阶段代码 |

### 高级功能（v4.0+）

| 功能 | 描述 |
|------|------|
| 🎯 自然数据嵌入 | 数据自然融入叙述，避免 AI 痕迹 |
| 🔍 深度对比分析 | 不仅列出对立观点，还分析分歧原因 |
| ✨ 综述润色 | 自动消除 AI 腔，让语言更干练 |
| 🔗 引用拆分 | 连续引用拆分为结构化陈述 |
| 💬 观点碰撞 | 结构化分析学术争议 |
| 📊 统计数据提取 | 从论文中提取 OR 值、P 值等统计数据 |

### 技术架构

```
┌─────────────┐
│   前端      │ React + TypeScript
│  localhost  │ http://localhost:3000
└──────┬──────┘
       │ HTTP API
       ▼
┌─────────────────────────────────────────┐
│              后端服务                    │
│  FastAPI + Python + PostgreSQL          │
│  ┌───────────────────────────────────┐  │
│  │  1. 题目分析（混合分类器）         │  │
│  │  2. 智能搜索（数据库+外部API）    │  │
│  │  3. 质量过滤（低质量文献过滤）     │  │
│  │  4. 综述生成（DeepSeek LLM）       │  │
│  │  5. 阶段记录（数据库追踪）         │  │
│  │  6. 论文入库（自动保存元数据）     │  │
│  └───────────────────────────────────┘  │
└──────────────┬──────────────────────────┘
               │
               ▼
┌─────────────────────────────────────────┐
│            外部服务                      │
│  • DeepSeek API（LLM生成）              │
│  • ScholarFlux（多数据源聚合）          │
│    - AMiner（中文文献）                 │
│    - OpenAlex（英文文献）               │
│    - Semantic Scholar（补充搜索）       │
│    - Crossref（引用数据）               │
│    - DataCite（研究数据集）             │
│    - arXiv（预印本论文）                │
│    - Google Scholar（补充搜索）         │
└─────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────────┐

                         │
         ┌───────────────┼───────────────┐

         │              │              │
         │              │         阶段1-4（共享）
         │              │              │
         ▼              ▼              ▼
    ┌─────────┐    ┌─────────┐    ┌─────────┐

    └─────────┘    └─────────┘    └─────────┘
```

| 功能 | 阶段 | 输出 | 用途 |
|------|------|------|------|

## 🎯 快速开始

### 使用 Docker（推荐）

```bash
# 1. 启动 PostgreSQL 数据库
docker-compose up -d

# 2. 配置环境变量
cd backend
cp .env.example .env
# 编辑 .env，填入 DEEPSEEK_API_KEY 和其他 API Keys

# 3. 安装后端依赖
pip install -r requirements.txt

# 4. 启动后端
python3 main.py

# 5. 启动前端
cd ../frontend
npm install
npm run dev

# 6. 访问系统
open http://localhost:3000
```

### 本地运行

```bash
# 1. 启动 PostgreSQL（或使用 Docker）
docker-compose up -d

# 2. 启动后端
cd backend
python3 main.py

# 3. 启动前端
cd frontend
npm run dev

# 4. 访问系统
open http://localhost:3000
```

### 环境配置

```bash
# 复制配置文件
cp backend/.env.example backend/.env

# 编辑配置，填入 API Keys
# - DEEPSEEK_API_KEY（必需）
# - AMINER_API_TOKEN（推荐，中文文献）
# - SEMANTIC_SCHOLAR_API_KEY（可选）

# 数据库配置
# - DB_TYPE=postgresql
# - DB_HOST=localhost
# - DB_PORT=5432
```

## 📊 Temperature 配置说明

系统针对不同任务类型优化了 DeepSeek 的 temperature 参数：

| 任务类型 | Temperature | 理由 |
|----------|-------------|------|
| 大纲生成 | 0.3 | 结构化输出需要稳定性 |
| 关键词提取 | 0.3 | 精确提取需要准确性 |
| 综述生成 | 0.4 | 平衡创造性与准确性 |
| 数据提取 | 0.3 | 数据提取需要准确 |
| 对比分析 | 0.3 | 批判性分析需要严谨 |
| 综述润色 | 0.4 | 适度优化保持原意 |

## 🎨 前端功能

### 主界面

| 组件 | 功能 |
|------|------|
