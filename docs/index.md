## 🚀 系统概览

PaperOverview 是一个基于 AI 的智能文献综述生成系统，用户只需输入论文题目，系统即可自动生成包含 50+ 篇参考文献的高质量文献综述。

### 核心功能

| 功能 | 描述 |
|------|------|
| 🔍 智能文献搜索 | 支持中英文文献，语言区分搜索（中文用 AMiner，英文用 OpenAlex） |
| 🤖 AI 综述生成 | 基于 DeepSeek 大模型，自动生成结构化综述 |
| ✅ 质量保证 | 自动过滤低质量文献，验证引用数量和年份分布 |
| 📊 参数可配 | 支持自定义文献数量、年份占比、英文占比等 |

### 技术架构

```
┌─────────────┐
│   前端      │ React + TypeScript
│  localhost  │ http://localhost:3000
└──────┬──────┘
       │ HTTP API
       ▼
┌─────────────────────────────────────────┐
│              后端服务                    │
│  FastAPI + Python + MySQL               │
│  ┌───────────────────────────────────┐  │
│  │  1. 题目分析（混合分类器）         │  │
│  │  2. 文献搜索（多数据源聚合）       │  │
│  │  3. 质量过滤（低质量文献过滤）     │  │
│  │  4. 综述生成（DeepSeek LLM）       │  │
│  │  5. 引用处理（排序、合并、过滤）   │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
       │
       ▼
┌─────────────────────────────────────────┐
│            外部服务                      │
│  • DeepSeek API（LLM生成）              │
│  • AMiner API（中文文献）               │
│  • OpenAlex API（英文文献）             │
│  • Semantic Scholar（补充搜索）         │
└─────────────────────────────────────────┘
```

## 🎯 快速开始

### 本地运行

```bash
# 1. 启动后端
cd backend
python3 main.py

# 2. 启动前端
cd frontend
npm run dev

# 3. 访问系统
open http://localhost:3000
```

### 环境配置

```bash
# 复制配置文件
cp backend/.env.example backend/.env

# 编辑配置，填入 API Keys
# - DEEPSEEK_API_KEY
# - AMINER_API_TOKEN
```
