## 配置项说明

### 基本配置（默认显示）

1. **目标文献数量** (target_count)
   - 默认值：50
   - 范围：10-100
   - 说明：综述中引用的文献总数

2. **近5年文献占比** (recent_years_ratio)
   - 默认值：0.5 (50%)
   - 范围：0.1-1.0 (10%-100%)
   - 说明：最近5年发表的文献占比

3. **英文文献占比** (english_ratio)
   - 默认值：0.3 (30%)
   - 范围：0.1-1.0 (10%-100%)
   - 说明：英文文献的占比

### 高级配置（点击"高级配置"展开）

4. **搜索年份范围** (search_years)
   - 默认值：10
   - 范围：5-30
   - 说明：搜索最近N年发表的文献

5. **最多搜索查询数** (max_search_queries)
   - 默认值：8
   - 范围：1-20
   - 说明：最多使用多少个搜索查询

## 使用方法

1. 访问 http://localhost:3000/
2. 在输入框中输入论文题目
3. 在下方的配置面板中调整参数
   - 拖动滑块调整数值
   - 或点击滑块直接跳到特定位置
4. 点击"✨ 生成综述"按钮

## 注意事项

- 配置参数在生成综述时生效
- 调整配置后无需刷新页面
- 生成进行中时配置面板会被禁用
- 高级配置默认折叠，点击可展开
